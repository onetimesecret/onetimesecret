This comprehensive guide combines the glossary of standardized term translations and language-specific translation notes for Ukrainian. It serves as the authoritative reference for maintaining consistency and quality when translating Onetime Secret content into Ukrainian.

## Introduction

This document merges two essential resources:

1. **Glossary** - Standardized translations of key terms, UI elements, and technical vocabulary

Use this guide to ensure translations are consistent, accurate, culturally appropriate, and grammatically correct.

## Critical Quality Guidelines

These are the most important rules to follow when translating Onetime Secret into Ukrainian:

| Rule | Correct | Incorrect | Example |
|------|---------|-----------|---------|
| **Brand Name: Onetime Secret** | Onetime Secret (do not translate) | ~~Одноразова таємна~~, ~~Одноразовий секрет~~ | ✓ "Ласкаво просимо до Onetime Secret"; ✗ "Ласкаво просимо до Одноразової таємної документації" |
| **Product Names** | Identity Plus, Global Elite, Custom Install (do not translate) | ~~Ідентичність Плюс~~, ~~Глобальна Еліта~~ | ✓ "Upgrade to Identity Plus"; ✗ "Перейти на Ідентичність Плюс" |
| **Translation of "secret" (general)** | таємний/таємна/таємне (adjective); таємниця (noun) | ~~секретний/секрет~~ (overly personal connotation) | ✓ "таємне повідомлення"; ✗ "секретне повідомлення" |
| **Secret Links** | таємні посилання (consistently) | ~~секретні посилання~~ (inconsistent with other translations) | ✓ "Що таке таємні посилання?"; ✗ "Що таке секретні посилання?" then "Як працюють таємні посилання?" |
| **Secret message** | таємне повідомлення | ~~секретне повідомлення~~, ~~секрет~~ | ✓ "Створити таємне повідомлення"; ✗ "Створити секрет" |
| **Password vs Passphrase** | пароль (for account login), парольна фраза or ключова фраза (for secret protection) | ~~пароль~~ for both cases | ✓ "Введіть ваш пароль для входу" and "Захистити парольною фразою"; ✗ "Захистити паролем" |

| User Address | Second person singular "ви" (avoid informal "ти" in documentation) | ✓ "Ви створюєте таємне повідомлення"; ✗ "Ти створюєш секретне повідомлення" |

## Ukrainian Grammar Specifics

### Gender Agreement

Ukrainian has three genders (masculine, feminine, neuter). Ensure adjectives agree:

| Noun | Gender | Example with "таємний" |
|------|--------|----------------------|
| повідомлення | neuter | таємне повідомлення |
| посилання | neuter | таємне посилання |
| таємниця | feminine | нова таємниця |
| вміст | masculine | таємний вміст |

### Cases

Use correct case forms depending on context:

- Nominative (називний): "таємне посилання існує"
- Genitive (родовий): "створення таємного посилання"
- Dative (давальний): "доступ до таємного посилання"
- Accusative (знахідний): "створити таємне посилання"
- Instrumental (орудний): "поділитися таємним посиланням"
- Locative (місцевий): "в таємному посиланні"

## Translation Guidelines

1. **Consistency**: Use the same translation for a term throughout the application
2. **Context**: Consider how the term is used in the application
3. **Cultural Adaptation**: Adapt terms for local context when necessary
4. **Technical Accuracy**: Ensure accurate translation of security terms
5. **Tone**: Maintain a professional yet direct tone
6. **Consistency above all**: Use the same translation for a term throughout the app
7. **Context matters**: "Таємниця" for noun, "таємний/а/е" for adjective
8. **Natural language**: Translations should sound natural in Ukrainian, not like a calque from English
9. **Technical precision**: Security terms require precise translation
10. **Don't translate brands**: Onetime Secret, Identity Plus, Global Elite, Custom Install remain in English
