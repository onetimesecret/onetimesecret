# Utils Scripts

General utility tools for locale file maintenance and analysis.

## Scripts
- `fix-json-unicode.py` - Fixes Unicode encoding issues in JSON files
- `search-key-usage` - Searches codebase for i18n key usage
- `generate-translation-template.sh` - Creates templates for new translations

## Usage
Utility scripts for various locale maintenance tasks.
Use search-key-usage to find unused i18n keys before cleanup.
