# Harmonize Scripts

Tools for synchronizing locale file structures with the base English locale.

## Scripts
- `fix-locale-conflicts` - Resolves type mismatches in a single locale file
- `fix-all-locale-conflicts` - Batch fix structural conflicts in all locales
- `harmonize-locale-file` - Synchronizes key structure with base locale
- `harmonize-locale-files` - Batch harmonize all locale files
- `github-action-harmonize.sh` - GitHub Actions integration for CI/CD

Usage: First run fix-locale-conflicts, then harmonize-locale-files.
