# Migrate Scripts

Tools for migrating locale keys and applying translations from external sources.

## Scripts
- `migrate-keys.py` - Migrates keys between different locale file versions
- `migrate-locale.py` - Full locale file migration with structure changes
- `apply-translations.sh` - Applies translations from external translation files

## Usage
Use these tools when refactoring i18n key structures or importing translations.
Python scripts require JSON module. Backup files before migration.
