# Locales Directory (DEPRECATED)

This directory is deprecated. Locale files are now generated to `generated/locales/`.

## Current Architecture

**Source of truth**: `locales/content/{locale}/*.json`
- Flat key-value format with metadata for translation management
- Version-controlled and human-editable

**Generated output**: `generated/locales/{locale}.json`
- Pre-merged JSON files consumed by frontend and backend
- Automatically generated by sync script

## Sync Workflow

```bash
# Generate merged locale files (runs automatically on pnpm dev/build)
pnpm run locales:sync

# Or use Python directly
python locales/scripts/sync_to_src.py --all --merged
```

## Translation Rules

1. Edit files in `locales/content/` - the source of truth
2. Use `en/` as reference - match the structure exactly
3. Preserve keys - only translate values
4. Keep placeholders intact: `{count}`, `{email}`, `{name}`

Security messages require special handling - see `SECURITY-TRANSLATION-GUIDE.md`.

## Testing

```bash
pnpm test:unit security-messages   # Validate security message compliance
pnpm run type-check                # Check JSON syntax
pnpm run i18n:generate-types       # Regenerate type definitions
```
