After clicking "Generate Password":

User would see a confirmation screen showing:

  • The generated password: xK9$mP2#vL5@
  • The one-time link to share
  • Two copy buttons: "Copy Link Only" and "Copy Both"
  • Tip: "The link shows the password—you don't need to share it separately"

(This state is not mocked yet, but would be similar to 04-confirmation.html with password displayed)

Feature Discovery Goal:

This feature is currently discovered by <10% of users. The Express Lane design aims to increase discovery to >40% by:

  • Prominent "or generate a random password →" link on main screen
  • Clear language (not buried in options)
  • Separate visual mode (not a checkbox in a form)
