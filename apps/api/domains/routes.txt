# apps/api/domains/routes
#
# Otto Routes - Domains API (Site-Only Endpoints)
# Using direct Logic class routing with authentication strategies
#
# Domains API handles custom domain management endpoints.
# These endpoints are not part of the public API and don't need versioning.
# All endpoints require authentication and organization context.
